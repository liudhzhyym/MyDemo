JPush SDK for iOS 简介
=======================
JPush SDK for iOS 是一套由极光提供的软件库，使开发者可将极光的推送服务集成到自己的 iOS app 中。

关于 JPush SDK for iOS 的集成，请查阅我们详细的集成指南和 API 指南。
也可参考下载的压缩包中附带的示例 demo 来通过代码学习本 SDK 的用法。


新手指南
--------
0. 从官方路径（https://docs.jiguang.cn/jpush/resources/）下载本 SDK。
1. 参考iOS SDK集成指南（https://docs.jiguang.cn/jpush/client/iOS/ios_guide_new/） 快速把 Demo 跑起来，体验 JPush 的服务。
2. 复制 .h和.a（extension的.h和.a库请复制到extension的target中） 文件资源文件到你项目对应的文件夹下，按照集成指南初始化SDK。
3. 根据你的业务来构建通知策略。基础 api 教程见 demo 实例，详细 api 解释见：https://docs.jiguang.cn/jpush/client/iOS/ios_api/
