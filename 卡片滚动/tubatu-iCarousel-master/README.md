# tubatu-iCarousel
这是高仿土巴兔选择装修风格的效果IOS版本，Android版本请点击这里[tubatu-viewpager](https://github.com/hanhailong/tubatu-viewpager)，这里通过自定义iCarousel来实现，为什么通过iCarousel来实现呢，一是iCarousel很容易实现切换动画效果，二是选择的那一项自动居中
